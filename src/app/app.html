Kata 6

Visual Boundary Enforcement

Try to style a child component from the parent. Then fix it using the component API instead of breaking encapsulation.

Failing attempt

The parent targets the child's internal .core element with a selector. It fails because the child's styles are isolated.

Expected accent border. Still default.

Correct solution via component API

The parent adds a class to the host, and the child opts into it using :host(.accent).

Accent border applied by the child itself.

Reflection trigger: Why is visual isolation treated like encapsulation, not convenience?
